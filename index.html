Arcana Auth: Social Login with PNP & Custom UI!

This simple HTML/CSS/JS Web3 app example demonstrates how Arcana Auth SDK's built-in, plug-and-play login UI or custom login UI can be used to onboard users.

It uses Parcel bundler.

Use model:

  1. Log in using one of the connect options listed below.
  2. Get information about the user, account, login status, if reconnect is possible, etc.
  3. Perform wallet operations using the Arcana wallet UI
  4. Logout
